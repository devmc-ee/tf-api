## Description

API gateway for multiple projects: 

### Resto: 
  * API for thaifood.ee contents
  * API for thaifood.ee admin panel

### Auth: 
  * API for auth 

## Tech stack
* [Nestjs](https://docs.nestjs.com/) framework (TS) powered with [Fastify](https://fastify.dev/)
* 
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
